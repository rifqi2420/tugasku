  • javaScript adalah bahasa pemrograman yang digunakan untuk membuat website lebih interaktif dan dinamis. Ia memungkinkan developer untuk menambahkan fungsi-fungsi seperti animasi, interaksi pengguna, dan manipulasi konten halaman web.
  • Python adalah bahasa pemrograman tingkat tinggi yang fleksibel dan serbaguna, dikembangkan oleh Guido van Rossum. Python dikenal karena sintaksnya yang mudah dibaca dan dipelajari, serta mendukung berbagai paradigma pemrograman seperti berorientasi objek, fungsional, dan prosedural.
  • HTML adalah singkatan dari HyperText Markup Language, dan merupakan bahasa markup standar yang digunakan untuk membuat halaman web. Bahasa ini digunakan untuk menentukan struktur dan konten halaman web, seperti teks, gambar, tautan, dan elemen lainnya.
  • PHP adalah singkatan dari "Hypertext Preprocessor," sebuah bahasa pemrograman server-side open source yang populer digunakan untuk pengembangan web
